# Roadmap

**Phase 1: Foundation (Week 1-2)**
- Setup repo, install deps (ElizaOS, OpenClaw via npm).
- Deploy sample ERC-8004 registry di Base testnet (pakai Foundry/Hardhat, ref EIP docs).
- Integrate ENS/Basenames untuk naming (Ethereum.org guides).
- Copilot task: "Generate Solidity for Identity Registry with ERC-721."

**Phase 2: MVP Build (Week 3-4)**
- Build frontend dashboard.
- Create first agent template di ElizaOS + OpenClaw (e.g., yield bot dengan EVM plugin).
- Integrate x402 untuk payments (middleware example dari x402.org).
- Test end-to-end: Deploy agent, execute dummy trade.
- Milestone: Internal demo, commit ke `dev/mvp`.

**Phase 3: Beta & Iteration (Month 2)**
- Add reputation tracking via ERC-8004.
- Integrate CDP Coinbase untuk agentic wallets.
- User testing: Invite beta testers via X (@bram0511 post teaser).
- Fix bugs, optimize gas (Base docs best practices).

**Phase 4: Launch & Scale (Month 3+)**
- Mainnet deploy di Base.
- Add advanced features: Multi-agent, custom plugins.
- Marketing: Hackathon submission (ETHGlobal style), partnerships dengan DeFi protocols.
- Long-term: Open-source contributions, community agents marketplace.
