# Frontend - React/NextJS Dashboard

## Tech Stack
- **Next.js 15** - React framework
- **Wagmi + Viem** - Wallet connection & on-chain interactions
- **RainbowKit** - Wallet connect UI

## Setup
```bash
npm install
npm run dev
```

## Features (Phase 1)
- Wallet connection via RainbowKit (Base Sepolia + Base Mainnet)
- Basic dashboard layout
- Agent marketplace placeholder

## Configuration
Copy `.env.example` to `.env` and fill in the required values.
