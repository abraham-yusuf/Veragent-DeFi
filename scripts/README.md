# Scripts - Deployment scripts (Hardhat/Foundry untuk Base testnet)

## Available Scripts
- `deploy-base-sepolia.sh` - Deploy contracts to Base Sepolia testnet

## Usage
```bash
# Deploy to Base Sepolia
./scripts/deploy-base-sepolia.sh
```

## Prerequisites
- Contracts compiled (`cd contracts && npx hardhat compile`)
- Environment variables set (see `contracts/.env.example`)
